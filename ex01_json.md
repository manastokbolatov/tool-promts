# Exercise 1: Get Structured JSON

## Prompt

```text
Return only valid JSON.
Use exactly this structure:

{
  "city": string,
  "date": string,
  "weather": string
}

Provide the weather for Astana today.
Do not include any explanation or additional text.
```

## Model Output

```json
{
  "city": "Astana",
  "date": "2026-07-15",
  "weather": "Sunny"
}
```

The output is valid JSON.

## Answer

A program needs JSON because it has a fixed structure that can be parsed automatically, unlike a regular sentence.
